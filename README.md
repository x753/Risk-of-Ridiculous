# Risk of Ridiculous v0.9.8
### Survive as long as you can in a custom map.
	- Relive your childhood
	- Stronger enemies spawn over time
	- Easter eggs to find
	- Probably doesn't work with other mods
	- Just survive, for now

## Installation
Drop the `Ridiculous` folder into `\BepInEx\plugins\`

Make sure you have `R2API 2.1.16` (or equivalent)
